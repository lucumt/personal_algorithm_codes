# 题目要求:
> 现有1元、3元、5元等不同面值的硬币，其数量不限，给定一个金额，要求算出凑齐该值所需的最小硬币数

# 解题思路
* 将不同面值的硬币各取一个放入数组中，利用循环遍历的方式实现硬币数量不限
* 硬币分为选和不选，选中时要在金额中减去该硬币的面值
* 对于选和不选要分别计算结果，取较小值
* 可用动态规划消除递归算法中涉及到的重复计算

# 实现代码

## Java实现

```java
public class CoinTest {

    private int[] coins = new int[]{1, 3, 5};

    @Test
    public void testFind() {
        int max = 23;
        int[] result = dpFind(max);
        for (int i = 0; i <= max; i++) {
            System.out.println(i + " ==> " + rcFind(i) + "  => " + result[i]);
        }
    }

    /**
     * 采用递归的方式实现
     *
     * @param value 要凑齐的金额值
     * @return 所需的最小硬币数
     */
    public int rcFind(int value) {
        if (value == 0) {
            return value;
        }
        int min = -1;//初始值，没有选择
        for (int i = 0; i < coins.length; i++) {
            if (value >= coins[i]) {//只有剩余价值不小于硬币面值时，才能选择
                //选择
                int a = rcFind(value - coins[i]) + 1;
                if (a < min || min == -1) {
                    min = a;
                }
                //不选的话，直接跳过
            }
        }
        return min;
    }

    /**
     * 采用动态规划方式，避免递归时的重叠子问题
     *
     * @param value
     * @return 数组，其值为凑齐当前下标值所需的最小硬币数
     */
    public int[] dpFind(int value) {
        int[] result = new int[value + 1];
        for (int i = 1; i < value + 1; i++) {
            result[i] = 10000;
        }
        for (int i = 1; i < value + 1; i++) {
            for (int coin : coins) {
                if (i >= coin) {
                    int a = result[i - coin] + 1;
                    if (a < result[i]) {
                        result[i] = a;
                    }
                }
            }
        }
        return result;
    }
}
```
 
## Go实现
```go
import "fmt"

const MAX = 100

var coins = [3]int{1, 3, 5}

func recFind(n int) int {
	if n == 0 {
		return 0
	}
	min := MAX //设置一个最大值
	for _, v := range coins {
		if n >= v {
			//选择
			a := recFind(n-v) + 1
			if a < min {
				min = a
			}
		}
	}
	return min
}

func dpFind(n int) []int {
	result := make([]int, n+1)

	for i := 1; i < n+1; i++ {
		result[i] = MAX
	}
	for i := 1; i < n+1; i++ {
		for _, v := range coins {
			if v > i { //硬币面值比要凑的值大，直接返回
				continue
			}
			a := result[i-v] + 1
			if a < result[i] {
				result[i] = a
			}
		}
	}
	return result
}

func main() {
	max := 23
	result := dpFind(max)
	for i := 0; i <= max; i++ {
		fmt.Println(i, " => ", recFind(i), " => ", result[i])
	}
}
```
